## Anime folder structure

```
ANIME FOLDER
└── {SHOW NAME}
    │
    ├── banner.jpg ( banner for the show )
    ├── image.png  ( image for the show )
    ├── eps
    │   ├── {EPISODE NUMBER}.mp4
    │   ├── {EPISODE NUMBER} (Filler).mp4
    ├── movies
    │   ├── {MOVIE NUMBER} {MOVIE NAME}.mkv
    ├── openings
    │   └── {OPENING NUMBER} {SONG NAME}.mp4
    └── endings
        └── {ENDING NAME} {SONG NAME}.mp4

Example:
------------
ANIME FOLDER
└── bleach
    │
    ├── banner.jpg ( banner for the show )
    ├── image.png  ( image for the show )
    ├── eps
    │   ├── 1.mp4
    │   ├── 2.mp4
    │   ├── ...
    │   ├── 30.mp4
    │   ├── 31.mp4
    │   ├── 32.mp4
    │   ├── 33 (Filler).mp4
    │   ├── 34.mp4
    │   └── 35.mp4
    ├── movies
    │   ├── 1 Memories of Nobody.mkv
    │   ├── 2 The DiamondDust Rebellion.mkv
    │   ├── 3 Fade to Black.mkv
    │   └── 4 Hell Verse.mkv
    ├── openings
    │   └── 1 asterisk.mp4
    └── endings
        └── 1 life is like a boat.mp4
```

> **Note**
> If an episode file name includes a "(Filler)"
> the episode will be marked as a filler
